Ho risolto così.

Scaricare il repo di Larry Finger da Github : https://github.com/lwfinger/rtlwifi_new.
Scompattare la cartella .zip e posizionarcivisi dentro con:

Codice: Seleziona tutto
    cd rtlwifi_new-master


Procedere con l'installazione installando prima gli header con:

Codice: Seleziona tutto
    # apt-get install -y linux-headers-"`uname -r`"


Build:

Codice: Seleziona tutto
    $ make


Installare:

Codice: Seleziona tutto
    # make install


Rimuovere e aggiungere modulo:

Codice: Seleziona tutto
    # modprobe -rv rtl8723be
    # modprobe -v rtl8723be ant_sel=1 


Se non va, se il segnale è ancora debole, provare con:

Codice: Seleziona tutto
    # modprobe -rv rtl8723be
    # modprobe -v rtl8723be ant_sel=2 


Se la soluzione funziona, la si può rendere definitiva così, creando il seguente file:

Codice: Seleziona tutto
    # touch /etc/modprobe.d/50-rtl8723be.conf


con al suo interno la sola riga:

Codice: Seleziona tutto
    options rtl8723be ips=0 ant_sel=2
