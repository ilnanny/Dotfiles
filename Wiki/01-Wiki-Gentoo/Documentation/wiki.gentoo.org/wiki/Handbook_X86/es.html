Este es el manual de Gentoo, un esfuerzo para centralizar la información sobre Gentoo Linux. Este manual contiene las instrucciones de instalación para realizar la instalación basándose en Internet y partes sobre cómo se trabaja con Gentoo y Portage.

Contents

  • 1 Instalar Gentoo
  • 2 Trabajar con Gentoo
  • 3 Trabajar con Portage
  • 4 Configuración de red en Gentoo

Instalar Gentoo

En esta parte el lector aprende como instalar Gentoo en un sistema.

Acerca de la instalación de Gentoo Linux
Este capítulo muestra el enfoque para la instalación que se sigue en este manual.
Elegir el medio correcto para la instalación
Es posible instalar Gentoo de muchas formas. Este capítulo explica cómo instalar Gentoo utilizando el CD de instalación mínimo.
Configurar la red
Para poder descargar el último código fuente, se necesita configurar la red.
Preparar los discos
Para poder instalar Gentoo, se necesitan crear las particiones necesarios. Este capítulo describe cómo particionar un disco para su uso futuro.
Instalar los ficheros de instalación de Gentoo
Las instalaciones de Gentoo funcionar mediante un fichero stage3. En este capítulo describimos cómo extraer el archivo stage3 y configurar Portage.
Instalar el sistema base de Gentoo
Después de instalar y configurar un stage3, se configura el sistema base de modo que esté disponible un entorno mínimo.
Configurar el núcleo Linux
El núcleo Linux es la parte central de todas las distribuciones. Este capítulo explica cómo configurar el núcleo.
Configurar el sistema
Se necesita editar algunos ficheros de configuración importantes. En este capítulo se ofrece una visión general de estos ficheros y una explicación de cómo proceder.
Instalar las herramientas del sistema
En este capítulo se seleccionan e instalan algunas herramientas importantes.
Configurar el cargador de arranque
En este capítulo se instala y configura el cargador de arranque apropiado.
Terminar la instalación
La instalación está casi terminada. Los últimos toques se documentan en este capítulo.

Trabajar con Gentoo

Introducción a Portage
Este capítulo explica los "simples" pasos que un usuario definitivamente necesita para saber cómo mantener el software de su sistema.
Ajustes USE
Los ajustes USE son un aspecto muy importante de Gentoo. En este capítulo, los lectores aprenderán a trabajar con ajustes USE y comprenderán cómo estos ajustes interactúan con su sistema.
Características de Portage
Descubra las características que tiene Portage, por ejemplo compilación distribuida, ccaché y muchas más.
Guiones de inicio
Gentoo utiliza un formato especial para sus guiones de inicio, entre otras características, permite decisiones basadas en dependencias y guiones de inicio virtuales. Este capítulo explica todos estos aspectos y cómo trabajar con estos guiones.
Variables de entorno
En Gentoo las variables de entorno se gestionan de forma muy sencilla. Este capítulo explica cómo hacerlo y también describe variables utilizadas frecuentemente.

Trabajar con Portage

Archivos y directorios
Para conocer Portage en profundidad, en primer lugar aprenda dónde almacena sus ficheros y datos.
Variables
Portage es completamente configurable a través de varias variables que se pueden definir en el fichero de configuración o como variable de entorno.
Mezclar ramas de software
Gentoo ofrece software separado en varias ramas dependiendo de su estabilidad y el soporte de arquitecturas. "Mezclar ramas de software" explica cómo estas ramas se pueden configurar y cómo saltarse esta separación de forma individual.
Herramientas adicionales
Portage viene con algunas herramientas extra que pueden hacer que la experiencia con Gentoo sea aún más positiva. Continúe leyendo para descubrir cómo utilizar dispatch-conf y otras herramientas.
Árbol de Portage personalizado
Este capítulo ofrede algunos consejos y trucos para utilizar un árbol Portage personalizado, cómo sincronizar únicamente las categorías necesarias, inyectar paquetes y mucho más.
Características avanzadas
Conforme avanza el tiempo, Portage evoluciona y madura más y más. Se incluyen constantemente características adicionales. Muchas de ellas solo las utilizan los usuarios más avanzados. Este capítulo entrará en los detalles de estas características específicas.

Configuración de red en Gentoo

Comenzar
Una guía para obtener de forma rápida una interfaz de red funcional en la mayoría de los entornos más comunes.
Configuración avanzada
Aquí aprenderemos cómo funciona la configuración. Este conocimiento es un prerequisito antes de continuar con la configuración de red moudular.
Configuración de red modular
Aprenda cómo elegir los diferentes clientes DHCP, poner en marcha "bonding", "bridging", VLANs y mucho más.
Redes inalámbricas (wireless)
Configurar Gentoo para redes inalámbricas
Añadir funcionalidad
Los usuarios más inquietos pueden añadir sus propias funciones a las herramientas de red.
Gestión dinámica
Para los usuarios de ordenadores portátiles o personas que quieran mover su equipo entre redes diferentes.
