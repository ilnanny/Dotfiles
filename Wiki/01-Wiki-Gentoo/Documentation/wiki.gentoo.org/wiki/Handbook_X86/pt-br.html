Este é o Manual do Gentoo, um esforço em centralizar as informações sobre o Gentoo Linux. Este manual contém as instruções de instalação para uma instalação baseada na Internet e prossegue em como trabalhar com o Gentoo e o Portage.

Contents

  • 1 Instalando o Gentoo
  • 2 Trabalhando com o Gentoo
  • 3 Trabalhando com o Portage
  • 4 Configuração de rede do Gentoo

Instalando o Gentoo

Nesta parte o leitor aprende como instalar o Gentoo em um computador.

Sobre a instalação do Gentoo Linux
Este capítulo introduz o método de instalação documentado neste manual.
Escolhendo a mídia de instalação correta
É possível instalar o Gentoo de muitas maneiras. Este capítulo explica como instalar o Gentoo usando o CD de instalação mínima.
Configurando a rede
Para ser capaz de baixar o código fonte mais recente, a rede precisará estar configurada.
Preparando os discos
Para ser capaz de instalar o Gentoo, as partições necessárias devem ser criadas. Este capítulo descreve como particionar um disco para uso futuro.
Instalando os arquivos de instalação do Gentoo
As instalações do Gentoo são feitas através de um arquivo stage3. Neste capítulo descrevemos como extrair o arquivo stage3 e como configurar o Portage.
Instalando o sistema Gentoo
Depois de instalar e configurar o stage3, o sistema base está configurado de modo que um ambiente mínimo estará disponível.
Configurando o kernel Linux
O kernel Linux é o núcleo de qualquer distribuição. Este capítulo explica como configurar o kernel.
Configurando o sistema
Alguns arquivos de configuração importantes precisam ser editados. Neste capítulo é dada uma visão geral desses arquivos bem como explicações sobre como proceder.
Instalando as ferramentas do sistema
Neste capítulo algumas importantes ferramentas são selecionadas e instaladas.
Configurando o gerenciador de boot
Neste capítulo o gerenciador de boot correto é instalado e configurado.
Finalizando a instalação
A instalação está agora quase completa. Os toques finais são documentados neste capítulo.

Trabalhando com o Gentoo

Uma Introdução ao Portage
Este capítulo explica os passos mais "simples" que o usuário definitivamente precisa saber para manter o software em seu sistema.
USE flags
As USE flags são um aspecto muito importante do Gentoo. Neste capítulo os leitores aprendem a trabalhar com as USE flags e entendem como as USE flags interagem com seus sistemas.
Recursos do Portage
Descubra os recursos que o Portage tem, tais como suporte a compilação distribuida, ccache e mais.
Sistema de Init script
O Gentoo usa um formato especial de initscript que, entre outros recursos, permite decisões baseadas em dependências e initscripts virtuais. Este capítulo explica todos esses aspectos e explica como utilizar esses scripts.
Variáveis de Ambiente
Com o Gentoo, as variáveis de ambiente podem ser facilmente gerenciadas. Este capítulo explica como fazer isso e também descreve as variáveis usadas com maior frequencia.

Trabalhando com o Portage

Arquivos e Diretórios
Para conhecer o Portage a fundo, primeiro aprenda onde ele armazena seus arquivos e dados.
Variáveis
O portage é totalmente configurável através de diversas variáveis que podem ser ajustadas em seu arquivo de configuração ou por variáveis de ambiente.
Misturando ramos de software
O Gentoo provê software separados em diversos ramos, dependendo de sua estabilidade e suporte a arquiteturas. "Misturando Ramos de Software" explica como esses ramos podem ser configurados e como sobrescrever essa separação individualmente.
Ferramentas adicionais
O Portage vem com algumas ferramentas extras que podem tornar a experiência com o Gentoo ainda melhor. Continue lendo para descobrir como usar o dispatch-conf e outras ferramentas.
Árvore customizada do Portage
Este capítulo mostra algumas dicas e truques sobre como utilizar uma árvore customizada do Portage, como sincronizar apenas as categorias necessárias, injetar pacotes e mais.
Recursos avançados
Com o passar do tempo, o Portage evolui e torna-se cada vez mais maduro. Recursos adicionais estão sendo continuamente adicionados - muitos desse apenas úteis para usuários mais avançados. Este capítulo entrará em mais detalhes sobre esses recursos específicos.

Configuração de rede do Gentoo

Iniciando
Um guia para rapidamente ter a rede funcionando nos casos de ambientes mais comuns.
Configuração avançada
Aqui aprendemos como a configuração funciona - isso é um pré-requisito antes de continuar com a configuração modular de rede.
Rede modular
Aprenda como escolher entre os diferentes clientes DHCP, configurar agregação, bridges, VLANs e mais.
Wireless
Configurando o Gentoo em redes wireless
Adicionando funcionalidades
Usuários aventureiros podem adicionar suas próprias funções às ferramentas de rede.
Gerenciamento dinâmico
Para usuários de notebooks ou aqueles que movem seu computador entre diferentes redes.
