Ceci est le manuel de Gentoo. Il centralise l'information sur Gentoo/Linux. Il contient les instructions pour une installation basée sur Internet et des parties sur l'utilisation de Gentoo et de Portage.

Contents

  • 1 Installer Gentoo
  • 2 Utiliser Gentoo
  • 3 Utiliser Portage
  • 4 Configuration du réseau

Installer Gentoo

Dans cette partie, le lecteur apprend comment installer Gentoo sur un système.

À propos de l'installation de Gentoo
Ce chapitre introduit l'approche utilisée dans ce manuel pour installer Gentoo.
Choisir le bon medium d'installation
Il existe différentes façons d'installer Gentoo. Ce chapitre explique comment installer Gentoo à partir du CD minimal d'installation.
Configurer le réseau
Afin de pouvoir télécharger le dernier code source, le réseau doit être configuré.
Préparer les disques
Pour être en mesure d'installer Gentoo, les partitions nécessaires doivent être créées. Ce chapitre explique comment partitionner un disque pour un usage ultérieur.
Installer les fichiers d'installation de Gentoo.
L'installation de Gentoo se fait via une archive d'étape 3. Dans ce chapitre nous expliquons comment extraire cette archive et configurer Portage.
Installer le système de base de Gentoo
Après avoir installé et configuré une archive d'étape 3, le système de base est configuré de sorte qu'un environnement minimal soit disponible.
Configurer le noyau Linux
Le noyau Linux est le cœur de toute distribution. Ce chapitre explique comment configurer ce noyau.
Configurer le système
Quelques fichiers de configuration importants doivent être édités. Dans ce chapitre, une vue d'ensemble des ces fichiers est fournie ainsi que des explications sur la manière de procéder.
Installer les outils système
Dans ce chapitre, quelques outils importants sont choisis et installés.
Configurer le chargeur de démarrage
Ce chapitre montre comment installer et configurer le chargeur de démarrage qui convient.
Finaliser l'installation
L'installation est maintenant presque terminée. Ce chapitre explique les dernières finitions à apporter.

Utiliser Gentoo

Une introduction à Portage
Ce chapitre explique les aspects « élémentaires » qu'une utilisateur doit absolument connaître pour maintenir le logiciel sur son système.
Options de la variable USE
Les options de la variable USE sont de première importance sous Gentoo. Dans ce chapitre, le lecteur apprend à se servir des options de la variable USE ainsi que leur interactions avec le système.
Fonctionnalités de Portage
Dans ce chapitre, le lecteur découvre les fonctionnalités de Portage telles que la compilation distribuée, le ccache et autres.
Les scripts d'initialisation
Gentoo utilise un format spécial de scripts d'initialisation qui, entre autres fonctionnalités, autorise les prises de décisions basées sur les dépendances et les scripts virtuels d'initialisation. Ce chapitre explique tous ces aspects et comment utiliser ces scripts.
Variables d'environment variables
Sous Gentoo, les variables d'environnement sont faciles à gérer. Ce chapitre explique comment le faire, et décrit les variables les plus couramment utilisées.

Utiliser Portage

Fichiers et répertoires
Pour connaître Portage en profondeur, commencez par apprendre où il range ses fichiers et ses données.
Variables
Portage est entièrement configurable via de nombreuses variables qui peuvent être définies dans des fichiers de configuration ou en tant que variables d'environnement.
Mélanger des branches de logiciel
Gentoo fournit un logiciel divisé en branches selon le niveau de stabilité et l'architecture prise en charge. Mélanger des branches de logiciel explique comment ces branches se configurent et comment outrepasser ces divisions pour un programme particulier.
Outils additionnels
Portage est fourni avec quelques outils supplémentaires qui facilitent l'utilisation de Portage. Découvrez comment utiliser dispatch-conf et d'autres outils.
Arbre de Portage personnalisé
Ce chapitre fournit quelques trucs et astuces pour utiliser un arbre de Portage personnalisé, comment en synchroniser les seules catégories nécessaires, injecter des paquets et plus encore.
Fonctionnalités avancées
Au fil du temps, Portage évolue et murit de plus en plus. Des fonctionnalités additionnelles sont introduites en permanence – beaucoup d'entre elles ne sont utilisées que par des utilisateurs expérimentés. Ce chapitre fournit des explications détaillées sur ces fonctionnalités spécifiques.

Configuration du réseau

Démarrer le réseau
Un guide pour démarrer rapidement l'interface réseau dans les environnements les plus courants.
Configuration avancée
Ici nous apprenons comment fonctionne la configuration – c'est un préalable nécessaire avant d'attaquer les modules réseau.
Les modules réseau
Gentoo fournit une configuration réseau très flexible. Vous apprendrez ici comment choisir votre client DHCP, mettre en place l'agrégation de liens, les ponts réseaux, les VLAN et autres.
Réseaux sans fil
Configurer les réseaux sans fils sous Gentoo
Ajouter des fonctionnalités
Si vous vous sentez l'âme aventureuse, ajoutez vos propres fonctions de gestion du réseau.
Gestion dynamique
Ce chapitre concerne les utilisateurs de portables ou ceux qui connectent leur ordinateurs à plusieurs réseaux.
